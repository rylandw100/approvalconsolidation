# Receipt Images

This folder contains receipt images for reimbursement requests.

## How to add images:

1. Add your receipt image files to this folder (e.g., `uber-receipt.png`, `hotel-receipt.jpg`)
2. Reference them in the code using the path `/receipts/filename.ext`

## Current receipts:

- `uber-receipt.png` - Used for Uber reimbursement requests
- `hotel-receipt.png` - Used for hotel reimbursement requests  
- `restaurant-receipt.png` - Used for restaurant reimbursement requests

## Supported formats:

- PNG (.png)
- JPG/JPEG (.jpg, .jpeg)
- WebP (.webp)

## Image recommendations:

- Recommended size: 159px width (height will scale proportionally)
- Format: PNG or JPG
- Keep file sizes reasonable for fast loading
